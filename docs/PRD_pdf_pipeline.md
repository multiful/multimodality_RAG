# 📌 PDF 페이지 레이아웃 분류 및 맞춤형 처리 파이프라인

## 1. 문제 정의 (Problem Statement)

> **"PDF 전체 페이지에 대해 고비용 변환(Docling/VLM)을 수행하는 것은 연산 자원 및 처리 속도 면에서 비효율적이다."**

* **해결 전략:** PDF 페이지의 성격(**이미지, 텍스트, 테이블**)을 사전에 핵심 분류하여, 각 요소에 최적화된 경량/고성능 처리 알고리즘을 분기(Routing) 적용함으로써 전체 시스템 성능 및 비용 최적화를 달성한다.

## 2. 데이터 요소별 처리 및 분석 구조 (Routing Architecture)

```
PDF Page
   │
   ▼
페이지 분류 (이미지 / 텍스트 / 테이블)
   │
   ├── 이미지 ──▶ VLM (Qwen3-8B)──────────────────────┐
   │                                         │
   ├── 텍스트 ──▶ BGE-M3 + BM25/KeyBERT ──▶ DB (기업 브랜드/기업명 검색 및 탐지)
   │                                         │
   └── 테이블 ──▶ ViT 영역감지 + TableFormer  │
        (마크다운/DataFrame 변환, 별도 보관)  │
                                              ▼
                                    기업의 이름 및 티커
                                    ├── 주가
                                    ├── 기업의 재무제표 → 핵심지표만 요약
                                    └── 관련 뉴스기사 → (추출 방식 미정)
                                              │
                                              ▼
                                LLM을 통한 투자 insight 제공
```

이미지·텍스트 분기의 결과가 DB에서 기업 브랜드/기업명 검색·탐지에 함께 쓰이고, 테이블 분기는 재무제표를 구조 보존한 채 별도로 변환해 이후 재무제표 요약 단계에 공급하는 구조다.

## 3. 파이프라인 세부 요소 및 적용 기술

### 🖼️ 1) 이미지 (Image Routing)

* **적용 기술:** `VLM`
* **채택 이유:**
  * 시각적 특징을 텍스트 공간으로 정형화하여 상장기업 및 브랜드를 식별

### 📝 2) 텍스트 (Text Routing)

* **적용 기술:** `주모델(BGE-M3 Dense) + 보조모델(BM25 / KeyBERT Sparse)`
* **채택 이유:**
  * **BGE-M3:** 다국어 및 문맥 의미 검색(Dense Vector)에 특화
  * **BM25 / KeyBERT:** 고유명사(티커, 기업명, 정확한 키워드) 보완 검색
  * **하이브리드 임베딩:** 의미 검색과 키워드 검색을 결합하여 RAG 검색 정확도 최대화

### 📊 3) 테이블 (Table Routing)

* **적용 기술:** `ViT 기반 영역 감지 + TableFormer (Docling Table Engine)`
* **채택 이유:**
  * 단순 OCR의 '행/열 구조 깨짐' 문제를 방지
  * 재무제표의 셀 병합, 다중 헤더, 숫자 데이터의 상하좌우 관계를 보존한 채 **마크다운(`|---|`) 및 Pandas DataFrame 포맷으로 완벽 변환**

## 4. DB 이후 처리 흐름

* 이미지(VLM)·텍스트(BGE-M3+BM25/KeyBERT) 분석 결과를 DB에서 검색해 **기업의 이름 및 티커**를 확정
* 확정된 기업 기준으로:
  * **주가** 조회
  * **재무제표** — 테이블 분기에서 변환된 마크다운/DataFrame을 핵심지표만 요약
  * **관련 뉴스기사** — 추출 방식 미정(Open Question)
* 위 정보를 종합해 **LLM을 통한 투자 insight 제공**
